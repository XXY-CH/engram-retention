# Experiments

This directory contains reproducible experiment entrypoints and configurations.

## Contents

- `train_synthetic.py` - Synthetic diagnostic runner for needle, XOR,
  dictionary, and module-drop ablations.
- `configs/` - YAML experiment configurations.

Generated result directories under `experiments/results/` are ignored by
default. Commit curated summaries in `docs/progress/` or publication figures in
`analysis/` when they become part of the research record.
