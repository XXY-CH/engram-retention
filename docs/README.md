# Research Documents

This directory is the research control surface for Engram Retention.

## Contents

- `methodology/` - Research protocol and experiment discipline.
- `progress/` - Chronological notes about proof and experiment milestones.
- `proofs/` - Formal proof drafts, assumption audits, and closure notes.
- `architecture_summary_and_deviations.md` - Current architecture summary and
  known deviations from source papers.
- `architecture_vulnerabilities.md` - Known weak points and risk analysis.
- `open-source-codex-application.md` - Prepared text for the Codex for Open
  Source application.

When code changes alter a research claim, update the relevant proof/progress
document in the same change or explicitly record the gap.
