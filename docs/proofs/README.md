# Proof Trail

This folder holds theorem drafts, proof audits, and closure notes for the
architecture.

## Reading Order

Start with:

- `proof_plan.md`
- `pdf_assumption_audit_2026-05-03.md`
- `25-general-llm-replacement-necessary-conditions.md`
- `15-proof-closure.md`

Then read the numbered proof files by topic. The numbering reflects research
iteration order rather than a final paper outline.

## Scope

The active proof position is conditional and bounded. The repository studies
whether small auxiliary memory paths can improve sparse long-context recall
under explicit budgets. It does not assert a universal LLM replacement.
